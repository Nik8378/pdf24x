# PDF24x – Free PDF Tools Online

PDF24x is a free, browser-based PDF tool suite. All processing happens locally in your browser — files are never uploaded to any server.

## Tools
- Image to PDF
- Image to WebP
- Compress PDF
- Merge PDF
- Split PDF
- PDF to JPG

## Tech Stack
- Next.js 16
- TypeScript
- Tailwind CSS v4
- pdf-lib
- pdfjs-dist
- Zustand
- @dnd-kit

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Deploy
Deploy instantly on [Vercel](https://vercel.com) — connect your GitHub repo and it deploys automatically.
